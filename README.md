# Social Media Impact on Teen Mental Health

## Overview

This data analysis project explores the relationship between social media usage and mental health outcomes among teenagers. It aims to identify patterns, correlations, and potential impacts through statistical analysis and visualization of relevant datasets.

## Objectives

- Analyze social media usage data (e.g., time spent, platforms used) among teens.
- Examine mental health indicators such as anxiety, depression, self-esteem, and sleep quality.
- Identify correlations between social media habits and mental health metrics.
- Provide insights and recommendations based on findings.

## Methodology

- Data cleaning and preprocessing using Python (Pandas, NumPy).
- Exploratory data analysis (EDA) with visualizations (Matplotlib, Seaborn).
- Statistical modeling (e.g., regression analysis) to test hypotheses.
- Reporting results in Jupyter Notebooks.

## Tools and Technologies

- Python for data analysis.
- Jupyter Notebook for interactive analysis.
- Tableau for data visualization and dashboards.
- Power BI for business intelligence and reporting.
- Git for version control.

## Expected Outcomes

- Insights into how social media influences teen mental health.
- Recommendations for parents, educators, and policymakers.
